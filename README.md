# 🔐 פורטל מפעיל — חדר בריחה

מיני-סייט פנימי לצוות המפעיל של מתחם חדרי הבריחה.

## 📁 מבנה הקבצים

```
/
├── index.html          ← דף ראשי + ניווט
├── elm-street.html     ← מדריך אלם סטריט (תדריך, חדרים, צ'קליסט)
├── hakatzia.html       ← מדריך הקצביה (ממתין לתוכן)
├── qa.html             ← מדריך Q&A טלפוני
├── css/
│   └── style.css       ← כל הסגנון (dark theme, RTL, רספונסיבי)
├── js/
│   └── main.js         ← לוגיקה: accordion, tabs, checklist, search
└── README.md
```

## 🚀 העלאה ל-GitHub Pages

1. צור repo חדש ב-GitHub
2. העתק את כל הקבצים לתוך ה-repo
3. לך ל-Settings → Pages → Branch: `main` → Save
4. האתר יהיה זמין ב: `https://[שם-משתמש].github.io/[שם-repo]`

## ✏️ עדכון תוכן הקצביה

לפתוח `hakatzia.html` ולהחליף את ה-placeholder עם התוכן המלא (אותו מבנה בדיוק כמו `elm-street.html`).

## 📱 תכונות

- RTL עברית מלאה
- תצוגה מותאמת למובייל ודסקטופ
- Accordion לשאלות Q&A + חיפוש חופשי + פילטור לפי קטגוריה
- Tabs בין קטגוריות תוכן
- צ'קליסט אינטראקטיבי (נשמר בדפדפן)
- אנימציות חלקות
