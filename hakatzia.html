🎛️ ציוד שליטה

כל כפתור מפעיל אלמנט ספציפי. זהו לוח הבקרה הראשי שלכם.
🟤
שלט חום
כפתור אחד. שולט על פתיחת דלת הכניסה. קחו אותו איתכם לפני שיכנסו השחקנים.

קופסה שחורה, פינה ימנית של חדר הבקרה. מספר שמוזן → מופיע על צג הלובי.

1
🏪 המעדניה
משימה 1חיבור תנור לחשמל
השחקנים מחברים התנור לחשמל → נשמעת מנגינה קצרה.
🎛️
הדליקו את האור בחדר.
משימה 2מפת הנתחים → קוד התנור
הנהלים הסניטריים מפנים למפת הנתחים על הקיר → קוד לפתיחת דלת התנור.

בתנור: שקיות בשר + שטר דולר. השטר מנחה לגשת לקופה.
משימה 3הקופה → רשימת תבלינים
"במקרה של תקלה יש לפתוח את הקופה ידנית" — פתיחה ידנית בצד האחורי.

בפנים: רשימת תבלינים. צנצנות מאחוריהם → בפנים מספרים.
משימה 4חיוג בטלפון → שקילת בשר
מקישים מספרים לפי הסדר בטלפון על הדלפק → שומעים הקלטה שמנחה לשקול חבילות בשר.
2
❤️ חדר הלבבות
🎭
הפחדת השייטל: בזמן השקילה — קחו שקית מחדר הבקרה, פתחו הדלת בפתאומיות: "הנה השייטל שביקשתם!" — חזרו מיד ונעלו.
מטרהלמצוא שלושה לבבות
❤️
לב 1
על מטחנת הבשר על השולחן.
❤️
לב 2
במגירה. לפתוח: 3 כפתורים בו-זמנית — 2 ליד ארון חשמל, 1 בצד השולחן.
❤️
לב 3
להוציא פקק מכיור → רמז → משוך ראשים תלויים בסדר → לב ייפול מהצלעות.
חידהחידת המקרר (מתקדמים)
סינר עם תמונה → תקוע סכין בשקע חשמל → פותח ארון חשמל.
שלט "מחסן" ליד הארון → למשוך למטה ולהסתכל בעינית.
שחקן אחד מכניס יד לארון; השני מדריך בעינית.
סדר המתגים → חתיכת בשר רקוב בפח.
לכבות אור → UV (קוטל יתושים) מדליק → חושף חיצים על האריזה = סדר המתגים.
מעברשקיות למקרר → פתיחת חדר הבא
השחקנים שמים שקיות בויטרינה.
🎛️
מצאו רגע הגיוני → לחצו "דלת מזווה". מוזיקה נעצרת, דלת לחדר הבא נפתחת.
2b
🔒 פתיחת המקרר — הכלוב
🎛️
דלת המקרר נפתחת → פתחו כלוב בשלט כחול → הקרינו סרטון: לחצו מתג 1.
"איציק, תן לי את הקורבן הכי טוב שלך."
פעולהקשירת הקורבן + סיבוב 180°
השחקנים קושרים אחד מהקבוצה לרתמות בכלוב. לאחר שקשור:
סגרו דלת הכלוב. הדריכו בכריזה אם נדרש.
העבירו וידאו ל-LED 2. לחצו מתג 2.
אחרי 7–8 שניות: לחיצה ארוכה על המתג האדום → סיבוב 180°. החזיקו עד סוף הסיבוב.
3
🧟 פרדריק — חדר הגופה
ℹ️
הקורבן נכנס לחדר 3. שאר השחקנים בחדר 2 — רואים גופה עם צינורות וצריכים לחבר לפי מדריך. כשנכון → הודעה: "תקלה במקרר". בפריזר רמז: ידית הברז לפתיחת הצד האחורי של המקרר → מעבר לחדר הקורבן.
שלב אשחרור הקורבן
מנואלה על הצידנית → חיבור וסיבוב → אור בבטן הבובה → קוד שחרור מהכלוב.
💡 להזכיר לשחקנים להוסיף סולמית (#) אם צריך.
שלב בהצידנית → עכבר → עין → מוט
צידנית בכלוב → מתחת לשאר → נפתחת: איבר ראשון + גבינה.
גבינה → מלכודת עכברים → עין. עין → ראש האישה → יד קופצת → מוט עם קרס.
שלב גהירך + ג'ינס + כרטיס עובד
מוט + מוט ארוך → משיכת תיבה על מסילה → ירך נופלת. לשים ירך במקום:
🎛️
כפתור 14 בשלט כחול → ג'ינס נופל → מכיסו: ארנק + כרטיס עובד → מכונת ההחתמות → דלת האחרון נפתחת.
3b
🪚 הפחדה + חדר מיקרו
הפחדהכניסה עם מסור
לבשו סינר + מסכת חזיר. קחו שלט כתום ומסור חשמלי מחדר 2.
דלת מסתובבת: כפתור בצד ימין למעלה. ⚠️ זהירות על האצבעות.
בצד הצר של הדלת: מתג לבן → פלאשים + קולות מסור.
בסיום: כבו מתג, סגרו דלת.
חדר מיקרושני האיברים האחרונים
זרוע: מאחורי וילון — שתי רגליים. להלביש כפכף על הרגל → היד תיפול מהתקרה.

ראש: מיקרוגל + ראש עם שיניים + פלייר. לעקור 3 שיניים → בכל שן: צבע. ללחוץ 3 כפתורים בצבעים בו-זמנית → מיקרוגל נפתח → ראש, האיבר האחרון.
סיוםכל האיברים → המשחק נגמר
כל האיברים מחוברים לגופה.
🏁
לחצו "סוף משחק".

לסמן כל פריט לפני כל קבוצה. הצ'קליסט נשמר אוטומטית בדפדפן.
